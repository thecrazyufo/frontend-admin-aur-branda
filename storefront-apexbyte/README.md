# ApexByte Soft — Storefront

**ApexByte Soft** is a corporate blue-white enterprise software storefront.

- **Site ID**: `brandC`
- **Default Port**: `3003`
- **Framework**: Astro v6 (SSR) + React 19 + TailwindCSS v4
- **Theme**: Corporate blue-white, Plus Jakarta Sans

## Development

```bash
npm install
npm run dev       # http://localhost:3003
npm run build     # Production build
```

## Environment

```env
PUBLIC_SITE_ID=brandC
PUBLIC_API_URL=http://localhost:8080/api
```

## Design System

- **Primary color**: `#0055B3` (deep corporate blue)
- **Accent color**: `#00B4D8` (cyan-sky)
- **Font**: Plus Jakarta Sans (Google Fonts)
- **Card style**: Left accent border (`card-accent`)
- **Footer**: Light `#F5F8FF` background (NOT dark navy)
